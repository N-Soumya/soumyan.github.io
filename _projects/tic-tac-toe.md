## Project Overview
[cite_start]Engineered an AI agent using the **Minimax algorithm with alpha-beta pruning** to achieve optimal gameplay in a Misere variant[cite: 18, 21].

### Key Technical Contributions
* [cite_start]**Resource Management**: Directly managed search tree complexity to ensure smooth performance on mobile hardware[cite: 21].
* [cite_start]**P2P Networking**: Implemented a networking layer via Bluetooth/WiFi Direct for serverless device-to-device play[cite: 22].
* [cite_start]**Data Protocols**: Utilized custom JSON protocols to synchronize game states and turn-based logic[cite: 22].
